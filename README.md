# Data visualizer

A website to visualize data using @kitware/vtkjs , react and Nodejs.
  


# Features

 - Surface & Volume rendering.
 - Interactive widget.
 -  Clipping the volume 
 - Volume Contour
- Multi-Slice Image
- Marching Cubes
- Interactive Cropping volume widget
 ## Controls
 - Rotation.
 - Change Viewing distance.
 - Independent Actor Setup.
 - Responsive Ui.
